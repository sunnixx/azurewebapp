EXAMPLES

This page already has jQuery added. Open your dev console(F12 on Windows, CMD+ALT+J on Mac) and try the following codes.

Saving Data using POST

	$.ajax({
	    url: 'http://datastore.asadmemon.com/path/to/data', 
	    type: 'POST', 
	    contentType: 'application/json', 
	    data: JSON.stringify({number:22,date: new Date()}),
	success:function(res){console.log(res);}
	});

Where /path/to/data is the path where you want to save your JSON data.

Pulling Data using GET

	$.get('http://datastore.asadmemon.com/path/to/data',function(res){
		console.log(res);
	});

Where /path/to/data is the path where you want to save your JSON data.
